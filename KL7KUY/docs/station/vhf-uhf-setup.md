# VHF/UHF Station Setup

## Mobile Radio
- **Model**: Yaesu FTM-500DR
- **Bands**: 2m/70cm
- **Power Output**: 50W VHF / 35W UHF
- **Features**:
  - C4FM/FM
  - GPS
  - APRS

## Handheld Radios
- **Yaesu FT5D**
  - Dual-band (2m/70cm)
  - C4FM/FM
  - GPS/APRS

- **Quansheng UV-K5/UV-K6(8)/UV-5R+**
  - Dual-band (2m/70cm)
  - Wideband receive
  - Portable operation

## Antennas
- **Base Station**: Diamond X30A
  - Gain: 5.5dBi (VHF), 8.0dBi (UHF)
  - Height: 30ft

- **Mobile**: Comet SBB-5NMO
  - NMO mount
  - Dual-band

- **Portable**: Signal Stick
  - Flexible whip
  - Great for SOTA/POTA
